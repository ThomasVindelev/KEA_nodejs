    We usually navigate websites by going through different pages, messaging someone, or making changes in online documents.

    By making use of different CRUD-features on websites we are simultaniously sending or retrieving data from a web server trhough requests.

    • Get

      This commonly used request is used to retrieve information from a server.

      Our client is making as get request when we, for example, want to visit a website.

    • Post

      Whenever we send a message through an online communication platform we are asking the server to store and send our message.

      This is a post request. We are not asking the server for information, but rather providing it.

    • Put

      The put request is used to update online ressources.

    • Delete

      Like the name suggests, the delete request asks the server to remove a certain element from it's ressources.

      For example this would be used to delete posts on Facebook.
